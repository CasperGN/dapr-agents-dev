Dapr Agents Helm Chart

A comprehensive Helm chart for deploying Dapr Agents with monitoring, observability, and AI agent tooling.

Install via Helm Repository

helm repo add dapr-agents https://caspergn.github.io/dapr-agents-dev/
helm repo update
helm install dapr-agents dapr-agents/dapr-agents \
  --namespace dapr-agents --create-namespace

Install via OCI Registry

helm install dapr-agents oci://ghcr.io/caspergn/charts/dapr-agents \
  --namespace dapr-agents --create-namespace

With a custom LLM API key

helm install dapr-agents dapr-agents/dapr-agents \
  --namespace dapr-agents --create-namespace \
  --set llm.apiKey=your-key
